# Vesuvius Scroll Surface Detection

Detect and segment papyrus layers from 3D X-ray volumes
for the Vesuvius Challenge.

## Data
Downloaded via Kaggle API:
- train_images/: 3D TIFF volumes
- train_labels/: partial voxel annotations

## Goal
Learn voxel-wise segmentation of papyrus layers
from volumetric CT data.
